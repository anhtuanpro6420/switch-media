## About:

Media Switch project

## Environment:

```bash
Make sure you install docker
```

## Scripts:

Build image before run: 

```bash
yarn run docker-build
```

Run development server:

```bash
yarn run docker-dev
```

Build code:

```bash
yarn build
```

Start code:

```bash
yarn start
```

Linting:

```bash
yarn run lint
```

Fix linting:

```bash
yarn run lint:fix
```
